# To-Do List App

**Prompt:** Build a simple to-do list app with a clean interface where I can add, check off, and delete tasks.

## Features
- Add new tasks
- Check off completed tasks
- Delete tasks
- Persistent storage using localStorage
- Clean, modern UI with gradient background

## How to Use
1. Open `index.html` in your browser
2. Type a task in the input field
3. Click "Add" or press Enter to add the task
4. Check the checkbox to mark tasks as complete
5. Click "Delete" to remove tasks

## Files
- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - JavaScript functionality
