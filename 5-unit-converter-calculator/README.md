# Calculator & Unit Converter

**Prompt:** Make a calculator app that can handle regular math but also convert units (miles to kilometers, Fahrenheit to Celsius).

## Features

### Calculator Tab
- Full arithmetic operations (+, −, ×, ÷)
- Clear button to reset
- Delete button to remove last digit
- Decimal point support
- Large, easy-to-read display
- Responsive button layout

### Unit Converter Tab
- Multiple conversion types:
  - **Length**: Miles ↔ Kilometers
  - **Temperature**: Fahrenheit ↔ Celsius
  - **Weight**: Pounds ↔ Kilograms
  - **Volume**: Gallons ↔ Liters
- Real-time conversion
- Switchable units
- Precision output (4 decimal places)
- Reset button

## How to Use

### Calculator
1. Click on number buttons to enter values
2. Click operator buttons (+, −, ×, ÷) to select operation
3. Click "=" to calculate result
4. Use "C" to clear or "←" to delete last digit

### Unit Converter
1. Select the conversion type from the dropdown
2. Enter a value in the "From" field
3. Select the unit to convert from
4. The result appears automatically in the "To" field
5. Click "Reset" to clear values
