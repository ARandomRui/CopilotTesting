# Portfolio Website

**Prompt:** Create a basic website that introduces me and includes a contact form at the bottom.

## Features
- Professional navigation bar
- Hero section with call-to-action
- About me section with skills
- Contact form with validation
- Responsive design
- Smooth scrolling navigation
- Form submission feedback

## Sections
1. **Navigation** - Links to navigate the page
2. **Hero** - Welcome section with introduction
3. **About** - Personal introduction and skills
4. **Contact** - Contact form to reach out
5. **Footer** - Copyright information

## How to Use
1. Open `index.html` in your browser
2. Update the content with your information
3. Fill out the contact form to test functionality
4. Customize colors and styles in `styles.css`

## Files
- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - JavaScript functionality and form handling
